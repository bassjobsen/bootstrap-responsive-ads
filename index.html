Bootstrap with responsive banner ads

About

This sample code (powered by W3Masters, mediaCheck and Twitter Bootstrap demonstrates how you can use responive banner ads with Twitter bootstrap or other responsive html5 solutions.

This example implements the Toppler strategy from Ad Formats & Showcase.

Please resize your browser window or load this page in Responsinator.com

Based on device (screenwidth / viewport) different formats of graphic banners are shown. Also the banner postion change. Banners of the following formats are used: 160x600, 728x90, 468x60 and 320x50 (width x height in pixels).

Only loads banners which will be used. Every banner load is a real view for your stats. Banners will not resized or hide by CSS.

The strategy use three breakpoints: 768, 480 and 320 pixels.
